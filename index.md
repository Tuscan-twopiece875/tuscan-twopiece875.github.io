## 🛠️ Step-by-step installation

Follow these steps to set up the mod and the progression guide on your Windows computer.

### Step 1: Install NeoForge
The Create: Aeronautics mod requires the NeoForge mod loader. Go to the official NeoForge website. Download the installer for your specific Minecraft version. Run the installer file and select the "Install Client" option. Click OK to finish the process.

### Step 2: Prepare your Minecraft folder
Open the Windows search bar. Type `%appdata%` and press Enter. Open the folder named `.minecraft`. Locate the folder named `mods`. If this folder does not exist, right-click inside the `.minecraft` folder, select "New", then "Folder", and name it `mods`.

### Step 3: Add the mod
Download the Create: Aeronautics mod file from the official source. Move this file into the `mods` folder you located in the previous step. Do not extract the file. Minecraft handles the file in its original format.

### Step 4: Add the progression guide
Download the progression guide provided in this repository. Place this file in a folder where you can easily find it, such as your Documents or Desktop folder. You may open the files in any standard web browser or PDF viewer to read the instructions while you play.

### Step 5: Launch the game
Open your Minecraft Launcher. Select the installation profile that corresponds to the NeoForge version you installed. Click "Play". Once the game loads, verify that the mod icon appears in the "Mods" menu.

## 🏗️ Learning the mechanics

The mod introduces many new parts. Success requires understanding how these parts connect.

### Building airships
Start by building a hull. You need different blocks to define the shape of your ship. Use the "Chassis" blocks to define the physical boundary of the ship. Connect a "Flight Controller" to the chassis. This block serves as the brain for your vessel. 

### Engines and propulsion
Flight requires power. You must construct engines to move propellers. The mod uses rotational force from the base Create mod. Connect your engines to the propellers using gearboxes or shafts. Watch the speed gauge. Too much torque can break your components if you rotate them too fast.

### Managing lift
Lift blocks determine how much weight your ship carries. Place these blocks inside your hull. If your ship feels heavy, add more lift blocks. Ensure you distribute the weight evenly so the ship stays level during flight. You can use ballast tanks to adjust your flight height while in the air.

## 💡 Troubleshooting common issues

If you encounter problems, check these items first.

*   Game crashes on launch: Ensure your NeoForge version matches the mod version. If you have other mods, they might conflict with Aeronautics. Try running the game with only the required mods.
*   Ship does not move: Check if your engine receives power. Ensure you anchor the ship to the ground before you start the assembly process.
*   Missing textures: This often happens if the mod files did not load correctly. Restart the game client to refresh the asset cache.
*   Low frame rate: Open your Minecraft settings. Lower your render distance and turn off complex lighting effects. The addition of moving structures consumes significant processing power.
